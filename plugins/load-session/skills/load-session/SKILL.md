# Load Session

## Overview

`save-session`으로 저장된 세션 파일을 자동으로 찾아 읽고, 작업 맥락을 복원한다. 사용자가 파일을 직접 지정할 필요 없이 가장 최근 세션을 자동으로 불러온다.

## When to Use

- 새 세션에서 이전 작업을 이어서 하고 싶을 때
- `/load-session` 명령 실행 시

## 동작 절차

### 1. 세션 파일 탐색

프로젝트 루트의 `sessions/` 디렉토리에서 MD 파일을 찾는다:

```bash
ls sessions/*.md | sort -r | head -5
```

파일이 없으면 사용자에게 "저장된 세션이 없습니다"라고 안내하고 종료한다.

### 2. 최근 파일 자동 선택

- 파일명이 `YYYY-MM-DD-HHmm-{토픽}.md` 형식이므로 **파일명 역순 정렬**로 가장 최근 파일을 자동 선택한다
- 선택된 파일명을 사용자에게 알려준다

### 3. 세션 파일 읽기

선택된 MD 파일의 전체 내용을 읽는다.

### 4. 맥락 복원

세션 파일의 내용을 바탕으로:

1. **메타 정보** 확인: 날짜, 프로젝트, 브랜치
2. **변경된 프로젝트** 파악: 관련 서브프로젝트 목록
3. **작업 내용** 숙지: 배경/내용/영향 범위
4. **주요 결정사항** 확인
5. **미완성 작업 / 다음 단계** 확인: 이어서 할 작업 목록
6. **재개 시 참고사항** 확인: 특별한 주의사항

### 5. 작업 재개 안내

사용자에게 다음을 요약하여 보고한다:

- 이전 세션 요약 (한 줄)
- 남은 작업 목록
- 재개 시 참고사항 (있는 경우)

그리고 "어떤 작업부터 이어서 진행할까요?" 라고 질문하여 작업을 시작한다.
