# load-session

이전 세션에서 저장한 작업 내용을 자동으로 불러와 작업을 이어서 진행합니다.

## 사용법

```
/load-session:load-session
```

## 동작 절차

1. `sessions/` 디렉토리에서 MD 파일을 수정시간 기준으로 탐색
2. 가장 최근 파일을 자동 선택
3. 세션 파일 읽기 및 맥락 복원
4. 이전 세션 요약, 남은 작업 목록, 참고사항 보고
5. "어떤 작업부터 이어서 진행할까요?" 질문으로 작업 시작

## 전제 조건

- `save-session` 플러그인으로 저장된 세션 파일이 `sessions/` 디렉토리에 있어야 합니다.
